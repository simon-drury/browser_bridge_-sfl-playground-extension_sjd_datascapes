# Supported LLMs

This document provides setup instructions for various LLMs that are compatible with the project.

## OpenAI

### Setup Instructions
1. Sign up for an API key from [OpenAI](https://beta.openai.com/signup/).
2. Install the OpenAI SDK using `pip install openai`.
3. Set your API key in your environment variables:
   ```bash
   export OPENAI_API_KEY='your-api-key'
   ```

## Claude

### Setup Instructions
1. Visit [Claude](https://claude.ai/) and request access.
2. Follow the instructions provided to obtain your API key.
3. Install the necessary SDK.

## Copilot

### Setup Instructions
1. Ensure you have an active GitHub Copilot subscription.
2. Install the GitHub Copilot extension in your IDE.
3. Authenticate using your GitHub account.

## Ollama

### Setup Instructions
1. Check the [Ollama website](https://ollama.com/) for access to their services.
2. Install the Ollama CLI with:
   ```bash
   curl -sSfL https://ollama.com/install.sh | sh
   ```
3. Follow the on-screen instructions to configure your environment.

## LM Studio

### Setup Instructions
1. Download LM Studio from the official website.
2. Follow the installation guide specific to your operating system.
3. Launch LM Studio and configure it with your desired models and settings.
